# Contributing to Tic-Tac-Toe PWA

## Development Workflow
This project follows a Test-Driven Development (TDD) workflow with AI-assisted development.

### Git & Commit Standards
- Follow conventional commit format: `<type>: <description>`
- See detailed workflow: [docs/GIT_WORKFLOW.md](docs/GIT_WORKFLOW.md)
- Maintain 80%+ test coverage on all changes

### Testing Requirements
- All tests must pass: `npm test`
- Coverage must meet 80% threshold: `npm run test:coverage`
- Write tests first (TDD approach)

### Phase-Based Development
The project is developed in phases. Current phase: **Phase 7 - Advanced Features**

See [docs/deepseek-chat-continue.md](docs/deepseek-chat-continue.md) for complete development history and current status.
